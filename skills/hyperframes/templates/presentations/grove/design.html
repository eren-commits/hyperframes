v — Guidelines

Sic et
non.

The rules are short. The grove softens the moment any are violated.

sic.

  • Use rust for one element per slide — the one ember.
  • Set every headline in italic Playfair.
  • Keep page margins generous — 144px, never less.
  • Use mono only for chrome labels and folio numbers.
  • Tint everything toward forest; never toward neutral grey.

non.

  • Introduce a second accent. Rust is the only ember.
  • Use sans-serif headlines — italics carry the page.
  • Apply shadows or gradients. Surfaces stay flat.
  • Use uppercase for body. Italic is emphasis.
  • Crowd the margin. The grove needs air.
