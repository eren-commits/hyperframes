v — Guidelines

Sic et
non.

The rules are short. The system breaks the moment any are violated.

sic.

  • Use hot pink for one element per slide — the thing remembered.
  • Set every headline in italic Instrument Serif.
  • Keep body type lowercase and 300 weight.
  • Run all chrome in JetBrains Mono — labels, metadata, page numbers.
  • Tint neutrals warm; never cool grey.

non.

  • Introduce a second accent. Pink is the only voice.
  • Use sans-serif for headlines — italics carry the page.
  • Apply shadows or gradients. Surfaces stay flat.
  • Use uppercase for body. Emphasis is italic.
  • Crowd the margin. Density is daylight.
