05 — RULES

YES.
NO.

The studio runs on these. Break any and the voltage drops.

YES.

  • Volt one element per slide — never two.
  • Set every headline in Anton, uppercase.
  • Run body in Inter 500. Never bold.
  • Keep all borders 1px sharp, zero radius.
  • Motion is fast (0.25–0.4s) and never bouncy.

NO.

  • Introduce a second accent. Volt is the only voice.
  • Use rounded corners or any radius greater than 0.
  • Apply shadows, gradients, or glows.
  • Set headlines in serif or lowercase.
  • Use easings with overshoot — Studio doesn't bounce.
