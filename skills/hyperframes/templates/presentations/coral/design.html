04 — Motion

Slow.
Deliberate.

Motion is unhurried. Long durations, gentle eases. The page should feel like a magazine page being turned — slow, intentional.

05 — Rules

YES.
NO.

A short list. The magazine softens the moment any are broken.

YES.

  • Set every headline in Bebas Neue, maximum size, uppercase.
  • Use coral for exactly one element per slide.
  • Keep all body type in Manrope 400. Never bold.
  • Place cream surfaces inside the noir canvas — never floating.
  • Use 1px hairlines only; no shadows or gradients.

NO.

  • Introduce a second accent. Coral is the only voice.
  • Set headlines in serif or italic. Bebas is the page.
  • Use pure black (#000) — always the warm noir.
  • Crowd the margin. The page wants breath.
  • Apply shadows. The page is flat by design.
