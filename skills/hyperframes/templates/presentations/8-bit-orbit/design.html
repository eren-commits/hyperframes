04 // MOTION

STEPPED.
NEVER SMOOTH.

Motion is quantized — every frame snaps. Use steps() easings to keep the 30Hz CRT feeling. Glow may pulse on a 1-second blink.

// EASING

steps(8)

Default. Eight discrete frames from A to B — no in-between. The page snaps like a sprite move. Use for entrances, transitions, hover states.

transition: transform .25s steps(8);
animation: blink 1.4s steps(2) infinite;
// TIMING

1.4s BLINK

The cursor blink — 1.4 seconds, two steps. Apply it to alerts, prompts, and anything waiting for input.

--dur-blink: 1.4s;
--dur-step:  .25s;
--dur-load:  3.0s;
05 // RULES

OK.
NO.

The CRT enforces these. Break any and the illusion collapses.

OK.

  • Render every border 3px solid, every shadow stepped, never blurred.
  • Use cyan for friendly signal, magenta for hostile or accent.
  • Set every face in Press Start 2P, VT323, or Silkscreen.
  • Quantize all motion with steps() easings.
  • Reserve glow text-shadow for the one element that matters.

NO.

  • Anti-alias borders or shadows. Hard pixels only.
  • Use rounded corners, gradients, or anti-aliased SVG.
  • Apply glow to body text — only to accent display.
  • Use smooth cubic-bezier easings — the CRT runs in steps.
  • Use pure black for the canvas — always the navy variant.
